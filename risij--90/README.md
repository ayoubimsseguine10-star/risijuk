# Risij.uk Static Blog - Classic Theme 2025

This project is a sophisticated, multi-page static blog created for Risij.uk. This version features a complete redesign with a timeless "Classic Blog" theme for 2025. It is built entirely with semantic HTML5 and modern CSS3, with no JavaScript or external frameworks. The blog's purpose is to provide high-quality, SEO-friendly content and analysis to drive traffic to the main Risij.uk website.

## Tech Stack

-   **HTML5:** Semantic markup for structure and SEO.
-   **CSS3:** For all styling, layout, and responsiveness.
    -   **Flexbox & CSS Grid:** Used for creating robust and modern responsive page layouts.
    -   **CSS Variables:** For a maintainable and consistent classic color scheme and typography.
    -   **Media Queries:** For a mobile-first responsive design.

## Features

-   **Classic & Timeless Theme:** A professional and content-focused user interface with a light background, charcoal text, and a sage green accent.
-   **New Typography:** Uses an elegant font pairing (Playfair Display for headings, Roboto for body) for excellent readability and a classic feel.
-   **Fully Responsive:** Mobile-first design that adapts flawlessly to all screen sizes.
-   **SEO Optimized:** Each page has unique, rephrased meta titles and descriptions. The content is structured with proper heading hierarchy and includes keyword-rich anchor text.
-   **Multi-Page Structure:** Includes a homepage, three sub-pages (Offers, About, Contact), and 12 individual article pages.
-   **Authoritative Content:** All text has been rewritten with a professional, analytical tone.
-   **Zero Dependencies:** 100% pure HTML and CSS. No JavaScript, frameworks, or external libraries are required.
-   **Cross-Browser Compatible:** Designed to work on all modern web browsers.
