
# Risij.uk Static Blog

This is a simple, static blog created for Risij.uk using only HTML and CSS. The blog is designed to be fast, responsive, and SEO-friendly.

## File Structure

- `index.html`: The main homepage of the blog.
- `[article-name].html`: Individual pages for each blog article (e.g., `aldi-offers.html`).
- `offers.html`: A sub-page with an overview of all supermarket offers.
- `about.html`: A sub-page with information about Risij.uk.
- `contact.html`: A sub-page with a contact form.
- `styles.css`: The complete styling for the entire website.

## Features

- **Responsive Design:** The layout is built with CSS Grid and Flexbox, ensuring a great experience on all devices.
- **SEO Optimized:** Semantic HTML, unique meta descriptions, and keyword-rich content are used throughout the site.
- **No Dependencies:** The blog is built with pure HTML and CSS, with no external frameworks or libraries.
- **Modern UI/UX:** The design is clean, modern, and user-friendly, with a professional color scheme and typography.

## How to Use

Simply open the `index.html` file in your web browser to view the homepage. From there, you can navigate to all other pages on the site.
